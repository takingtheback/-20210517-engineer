# 슬기로운 우리동네생활
- 본인이 살고 있는 동네에서 자유로운 글들과 정보글을 공유할 수 있음
- 메인페이지에서 송파구에 관련된 정보들을 확인할 수 있음

>> 담당 기능  
- 주리님 : 동네 정보 Open API(Main Page)  
- 낙원님 : 질문 게시판 CRUD  
- 성현님 : 사진 게시판 CRUD  
- 재현님 : 회원가입
